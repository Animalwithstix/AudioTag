# AudioTag

AudioTag is a native macOS audio tagging app — batch metadata editing, renaming and organizing,
artwork management, and online lookups for large music libraries.

## Data & Service Credits

AudioTag is built on the shoulders of these excellent services:

- **BPM and key data provided by [GetSongBPM](https://getsongbpm.com)**
- Music metadata from [MusicBrainz](https://musicbrainz.org)
- Cover art from the [Cover Art Archive](https://coverartarchive.org) and [fanart.tv](https://fanart.tv)
- Release data from [Discogs](https://www.discogs.com)
- Audio identification by [AcoustID](https://acoustid.org)
